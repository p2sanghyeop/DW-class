static  
static 목적: 공유(share)  
static으로 선언된 메소드와 변수는 인스턴스화를 하지 않아도 호출이 가능

공유는 하되 변하지 않는것에 사용한다(질병코드, 진단코드)

변수종류  
static 변수

non-static 변수

싱글톤 패턴  
코딩 디자인 패턴중 하나  
싱글톤은 스태틱 개념을 응용함

주로 공통된 객체(클래스)를 여러개 생성해서 사용하는 상황에서 많이 사용

싱글톤은 여러차례 호출되더라도 실제로 생성된느 객체(클래스)는 하나다

static을 사용하는 경우

1. 코드를 정의할때  
단 해당클레스에는 변수만 존재
2. 객체를 여러개 생성해서 사용하는 상황일때(싱글톤 패턴)  
ex)데이터베이스 연동할때
