get, set 메소드  
필드변수 접근지정자를 private으로 설정함으로써  
다른 클래스파일에서 불러올수 없음(보안성)

1. private 필드변수를 생성자로 접근
    장점) 클래스호출과 동시에 값 초기화
    단점) 유지보수 힘듬   
    (필드변수가 많아지면 많아질수록 상황 악화)
2. 필드변수마다 메소드를 생성  
    장점) 유지보수 편리  
    단점) 필드변수가 추가될때마다 메소드를  
    만들어야함
    -> 이클립스에서 자동으로 생성해줌

파일을 호출할때는 new를 사용한다
