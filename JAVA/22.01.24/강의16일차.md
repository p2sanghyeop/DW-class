1. 클래스는 필드변수(전역변수)와 메소드로 구성
2. 필드변수 앞에는 접근지정자
    (public, private, protected, default public)
3. 자바에서 클래스는 객체
4. 객체는 고유한 특성과 행동을 가짐
    ex) 학교
    특성
    1. 학생수
    2. 교직원수
    3. 반 수   
    
    행동
    1. 수업을 듣다
    2. 선생한테혼나다.
    3. 교장의지루한말을듣는다
5. 객체를 하는이유
    1. 업무 분담을 위해
    2. 일의 효율성을 위해서
